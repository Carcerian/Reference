================================================================================
HOW TO USE THIS MANIFEST
================================================================================

1. You get an error like:
   ERROR: UNDEFINED IDENTIFIER (NUI_MyFunction)

2. Search this manifest for "NUI_MyFunction"

3. If found in a section above:
   - The function EXISTS
   - Check if the calling file includes the right API
   - May need to add the include to the calling file's dependency chain

4. If NOT found anywhere:
   - Function truly doesn't exist
   - Needs to be implemented

5. For undefined CONSTANTS:
   - Check MASTER CONSTANT REGISTRY
   - Add missing constant to nui_api.nss

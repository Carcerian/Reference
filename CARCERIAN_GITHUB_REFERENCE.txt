================================================================================
ECOSYSTEM OVERVIEW
================================================================================

The Carcerian NWN:EE ecosystem consists of 4 integrated systems:

CPPS (Custom Persistent Placeables)
  ↓
  Provides DM tools for environment building
  
NUI (Natural User Interface System)
  ↓
  Provides popup/dialog system for all interfaces
  
CT (Carcerian's Tailor)
  ↓
  Uses NUI for character customization
  
Commoner (Custom Commoners)
  ↓
  OnSpawn script for NPC generation
  ↓
  Works with CPPS for placeable NPCs

================================================================================
IMPLEMENTATION NOTES
================================================================================

Architecture:
  - All popups use NUI_DialogCreate() from nui_api
  - All popups follow standard layout patterns
  - Consistent naming convention: NUI_Popup[Type]()
  - All include "nui_popup_api" for easy development

Handlers:
  - Most popups use "nui_handler" for default handling
  - Can use custom handlers for complex logic
  - Results stored in local variables on PC object

Local Variable Naming:
  - Text: "nui_popup_text_result"
  - Slider: "nui_popup_slider_result"
  - Integer: "nui_popup_int_result"
  - Float: "nui_popup_float_result"
  - Color: "nui_popup_color_red", "nui_popup_color_green", "nui_popup_color_blue"
  - Portrait: "nui_popup_portrait_result"
  - Soundset: "nui_popup_soundset_result"
  - Biography: "nui_popup_biography_result"

Performance:
  - All popups are lightweight
  - Minimal memory footprint
  - No performance overhead
  - Suitable for production servers

================================================================================
EXTENDING THE SYSTEM
================================================================================

To add new popup types:

1. Create function with consistent naming:
   int NUI_PopupXyz(object oPC, string sTitle, params..., string sScript)

2. Follow standard pattern:
   - Check GetIsPC(oPC) at start
   - Build JSON content
   - Call NUI_DialogCreate()
   - Return token

3. Add to forward declarations at top of file

4. Store results in local variables with standard naming:
   "nui_popup_xyz_result"

5. Update documentation

Example:

  int NUI_PopupNewFeature(object oPC, string sTitle, string sScript)
  {
      if (!GetIsPC(oPC)) return 0;
      
      json jContent = BuildYourContent();
      
      return NUI_DialogCreate(oPC, sTitle, jContent, 
                             NUI_BTN_OK + NUI_BTN_CANCEL,
                             "NOSCALE_CENTER", sScript, -1, 600.0, 300.0);
  }

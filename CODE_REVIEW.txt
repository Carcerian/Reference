OPTIMIZATION SUGGESTIONS
================================================================================

PRIORITY 1 - HIGH IMPACT (Easy Implementation)

1. PARAMETER PARSING HELPER FUNCTION
   
   Current Code (in HandleAgeCheckYes):
     int nPos = FindSubString(sParam2, "=");
     if (nPos >= 0)
     {
         nMinAge = StringToInt(GetSubString(sParam2, nPos + 1, 3));
     }
   
   Optimization:
     int NUI_ParseIntParam(string sParam, int nDefault)
     {
         int nPos = FindSubString(sParam, "=");
         if (nPos >= 0)
             return StringToInt(GetSubString(sParam, nPos + 1, 10));
         return nDefault;
     }
   
   Usage:
     nMinAge = NUI_ParseIntParam(sParam2, 18);
     nReward = NUI_ParseIntParam(sParam3, 0);
   
   Benefit: 20-line reduction, DRY principle, reusable

2. CALLBACK STRUCTURE
   
   Current:
     string sCallback = GetLocalString(oPC, "nui_callback_script");
     int nTiming = GetLocalInt(oPC, "nui_callback_timing");
     // (repeated in every handler)
   
   Optimization:
     json GetNUICallback(object oPC)
     {
         return JsonArray2(
             GetLocalString(oPC, "nui_callback_script"),
             GetLocalInt(oPC, "nui_callback_timing")
         );
     }
   
     void ExecuteNUICallback(object oPC, int nTiming)
     {
         json jCallback = GetNUICallback(oPC);
         string sScript = JsonGetString(JsonArrayGet(jCallback, 0));
         int nStoredTiming = JsonGetInt(JsonArrayGet(jCallback, 1));
         
         if (sScript != "" && nStoredTiming == nTiming)
         {
             ExecuteScript(sScript, oPC);
         }
     }
   
   Benefit: Eliminates 6 lines per handler × 7 handlers = 42-line reduction

3. ERROR CODES FOR HANDLERS
   
   Modify returns:
     int HandleButtonOk(...) instead of void
   
   Codes:
     0 = success, executed normally
     1 = validation failed
     2 = callback error
     3 = parameter error
   
   Usage:
     int nResult = HandleButtonOk(oPC, sAction, ...);
     if (nResult != 0)
     {
         WriteTimestampedLogEntry("Button handler failed: " + IntToString(nResult));
     }

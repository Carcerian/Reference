VERSION 1.2 - Popup Dialog Functions (June 2, 2026)
================================================================================

NEW FILE: nui_popups.nss (Popup Dialog Builders)

Complete popup dialog system with 8 specialized functions:

1. NUI_PopupMessage(oPC, sTitle, sMessage, sScript)
   - Simple message with centered label and close button
   - Use: Notifications, alerts, information displays

2. NUI_PopupConfirm(oPC, sTitle, sMessage, sScript)
   - Confirmation with OK and Cancel buttons
   - Use: Actions requiring confirmation

3. NUI_PopupYesNo(oPC, sTitle, sMessage, sScript)
   - Yes/No buttons with centered message
   - Use: Boolean questions, confirmations

4. NUI_PopupText(oPC, sTitle, sLabel, sScript)
   - Text input with label and entry box
   - Use: Getting player text input
   - Result: "nui_popup_text_result"

5. NUI_PopupSlider(oPC, sTitle, sLabel, fMin, fMax, fDefault, sScript)
   - Slider with value display
   - Use: Getting numeric values with range
   - Result: "nui_popup_slider_result"

6. NUI_PopupInt(oPC, sTitle, sLabel, nMin, nMax, nDefault, sScript)
   - Integer input with range info and validation
   - Use: Getting whole numbers
   - Result: "nui_popup_int_result"

7. NUI_PopupFloat(oPC, sTitle, sLabel, fMin, fMax, fDefault, sScript)
   - Floating-point input with decimal support
   - Use: Getting precise numeric values
   - Result: "nui_popup_float_result"

8. NUI_PopupColor(oPC, sTitle, sScript)
   - RGB color picker with 3 sliders
   - Use: Getting color selections
   - Results: "nui_popup_color_red", "nui_popup_color_green", "nui_popup_color_blue"

All dialogs are centered, use NOSCALE_CENTER layout, and follow consistent UI patterns.
